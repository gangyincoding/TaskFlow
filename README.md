# 📋 TaskFlow

一个简洁高效的待办事项管理应用，让任务管理变得轻松愉快。

## ✨ 功能特性

- ✅ **任务管理** - 快速添加、完成、删除任务
- 💾 **本地存储** - 使用 localStorage 持久化保存数据，无需注册账号
- 🎨 **美观界面** - 现代化渐变设计，简洁直观
- 🎬 **流畅动画** - 任务添加、完成、删除都有平滑过渡效果
- 📊 **实时统计** - 一目了然地查看总计、进行中、已完成的任务数量
- 🔍 **智能筛选** - 按"全部"、"进行中"、"已完成"筛选任务
- 📱 **响应式设计** - 完美适配手机、平板和桌面设备
- 🎯 **优化滚动** - 固定高度 + 自定义滚动条，界面整洁不杂乱
- 🔒 **安全防护** - HTML内容转义，防止XSS攻击

## 🚀 快速开始

### 在线使用

直接在浏览器中打开 `index.html` 文件即可使用，无需任何配置或服务器。

### 本地部署

1. 克隆或下载项目
```bash
git clone <repository-url>
cd TaskFlow
```

2. 在浏览器中打开 `index.html`
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

就这么简单！TaskFlow 是一个纯前端应用，无需安装依赖或构建步骤。

## 📖 使用说明

### 添加任务
1. 在输入框中输入任务内容（最多100字符）
2. 点击"添加任务"按钮或按 `Enter` 键
3. 任务会立即出现在列表顶部

### 完成任务
- 点击任务前的圆形复选框
- 已完成的任务会显示删除线和绿色背景

### 删除任务
- 点击任务右侧的"删除"按钮
- 任务会以动画效果滑出并从列表中移除

### 筛选任务
- **全部** - 显示所有任务
- **进行中** - 仅显示未完成的任务
- **已完成** - 仅显示已完成的任务

### 查看统计
顶部统计卡片实时显示：
- 总计：所有任务数量
- 进行中：未完成任务数量
- 已完成：已完成任务数量

## 💡 核心功能实现

### 数据持久化
使用 `localStorage` 保存任务数据，数据格式：
```javascript
{
  id: timestamp,
  text: "任务内容",
  completed: false,
  createdAt: "ISO时间戳"
}
```

### 事件委托
使用事件委托优化性能，所有任务的点击事件都在父元素 `taskList` 上监听，避免为每个任务单独绑定事件。

### 安全性
通过 `escapeHtml()` 方法对用户输入进行 HTML 转义，防止 XSS 攻击。
